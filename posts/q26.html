Problem:-

Write a program in C to find the area and perimeter of a triangle whose sides are entered through the keyboard, with validation check.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators
  • C Conditional

The program to find area and perimeter of the triangle with validation check.

                    
                        #include <stdio.h>
                        #include <math.h>
                        int main() {
                        float s1, s2, s3, sp, area, per;
                        printf("Enter the sides of triangle:- ");
                        scanf("%f %f %f", &s1, &s2, &s3);
                        //Calculating area and perimeter of the triangle with validation check
                        if ((s1 + s2 > s3) && (s1 + s3 > s2) && (s2 + s3 > s1)) 
                        {
                        per = s1 + s2 + s3;
                        sp = per / 2;
                        area = sqrt(sp*(sp-s1)*(sp-s2)*(sp-s3));
                        }
                        else
                        printf("Invalid Triangle");
                        printf("Area and perimeter of triangle are:- \nArea = %f \nPerimeter = %f", area, per);
                        return 0;
                        }
