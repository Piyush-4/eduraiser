Program to print Trigonometric Ratios
Eduraiser!

Problem:-

If value of an angle is input through the keyboard, write a program to print all its Trigonometric ratios.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The Program to print Trigonometric Ratios

                    
                        #include <stdio.h>
                        #include <conio.h>
                        #include <math.h>
                        int main() 
                        {
                        float ang, sin_f, cos_f, tan_f, cosec_f, sec_f, cot_f;
                        const float PI = 3.141592;
                        printf("Enter the angle:- ");
                        scanf("%f", &ang);
                        sin_f = sin(ang*PI/180);
                        cos_f = cos(ang*PI/180);
                        tan_f = tan(ang*PI/180);
                        cosec_f = 1/sin_f;
                        sec_f = 1/cos_f;
                        cot_f = 1/tan_f;
                        printf("The Trigonometric ratios are:-\n sin%f = %f\n cos%f = %f\n tan%f = %f\n cosec%f = %f\n sec%f = %f\n cot%f = %f", ang, sin_f, ang, cos_f, ang, tan_f, ang, cosec_f, ang, sec_f, ang, cot_f);
                        getch();
                        return 0;
                        }
