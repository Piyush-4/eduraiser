Program to interchange values of two variables without using third variable
Eduraiser!

Problem:-

Write a program in C to interchange values of two variables. Without using third variable.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to interchange the values of two variables without using third variable

                    
                        #include <stdio.h>
                        int main() {
                        int a, b;
                        printf("Enter values for variables a and b:-  ");
                        scanf("%d %d", &a, &b);
                        printf("The values of variables are \na = %d \nb = %d", a, b);
                        //Interchanging values of variables
                        a = a + b;
                        b = a - b;
                        a = a - b;
                        printf("\nChanged values of variables are \na = %d \nb = %d", a, b);
                        return 0;
                        }
