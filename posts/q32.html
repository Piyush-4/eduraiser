Program to convert the distance given in km in mile, yard, feet and inches
Eduraiser!

Problem:-

Write a program in C to convert the distance given in km in mile, yard, feet and inches.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to convert the distance given in km in mile, yard, feet and inches

                    
                        #include <stdio.h>
                        int main() {
                        float km, cm;
                        long int mile, yard, feet, inch;
                        printf("Enter the distance in km:- ");
                        scanf("%f", &km);
                        //converting distance form km to mile, yard, feet and inches
                        cm = km * 1000.0 * 1000.0;
                        inch = cm / 2.54;
                        feet = inch / 12;
                        inch = inch % 12;
                        yard = feet / 13;
                        feet = feet % 3;
                        mile = yard / 1760;
                        yard = yard % 1760;
                        printf("\nDistance = %ld mile %ld yard %ld feet %ld inch", mile, yard, feet, inch);
                        return 0;
                        }

Output

                        Distance = 80 mile 528 yard 1 feet 4 inch
