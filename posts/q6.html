Problem:-

Write a program in C to find average of five numbers given by the user.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to find average of five numbers given by the user.

                #include <stdio.h>
                    int main() {    
                    
                        int num1, num2, num3, num4, num5;
                        float average;
                        printf("Enter five integers: ");
                        scanf("%d %d %d %d %d",  &num1, &num2, &num3, &num4, &num5);
                    
                        // Calculating average of five numbers
                        average = (num1 + num2 + num3 + num4 + num5) / 5;   
                        
                        printf("Average of given five numbers = %f", average);
                        return 0;
                    }

Output

                        6
                        7
                        8
                        Average of given five numbers = 6.000000
