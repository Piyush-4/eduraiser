Program to check the number is odd or even
Eduraiser!

Problem:-

Write a program in C to check that the number entered by the user is odd or even.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators
  • C Conditional

The program to check the given number is odd or even.

                    
                        #include <stdio.h>
                        int main() {
                        int num;
                        printf("Enter a Number:-  ");
                        scanf("%d", &num);
                        //Using if else statement to check the number is odd or even
                        if (num % 2 == 0)
                        printf("%d is even", num);
                        else
                        printf("%d is odd", num);
                       return 0;
                        }

Output

                    Enter a Numbers:- 35
                        35 is odd
