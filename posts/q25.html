Problem:-

Given the length and breadth of a rectangle, write a program in C to find whether the area of the rectangle is greater than its perimeter.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators
  • C Conditional

The program to find whether the area is greater than perimeter

                    
                        #include <stdio.h>
                        int main() {
                        int l, b, area, peri;
                        printf("Enter a Number:-  ");
                        scanf("%d %d", &l, &b);
                        //Calculating area and perimeter of the rectangle
                        area = l * b;
                        peri = 2 * (l + b);
                        if (area > peri)
                        printf("Area of rectangle is greater than its perimeter");
                        else
                        printf("Perimeter of rectangle is greater than its area");
                       return 0;
                        }

Output

                    Enter a Number:- 5
                        4
                        Area of rectangle is greater than its perimeter
