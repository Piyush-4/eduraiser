Problem:-

Write a program in C to find the total marks, average marks and C.G.P.A of a student using else if ladder. The marks obtained in five subjects will be entered through the keyboard.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators
  • C Conditional
  • C Logical operator

The program to find total marks, average marks and C.G.P.A of a student

                    
                        #include <stdio.h>
                        int main() {
                        int s1, s2, s3, s4, s5, total;
                        float avg;
                        printf("Enter marks obtained in five subjects:- ");
                        scanf("%d %d %d %d %d", &s1, &s2, &s3, &s4, &s5);
                        //Finding total marks, average marks and C.G.P.A of the student
                        total = s1 + s2 + s3 + s4 + s5;
                        printf("Total marks = %d\n", total);
                        avg = total / 5.0;
                        printf("Average marks = %f\n", avg);
                        if (avg >= 90) 
                        printf("C.G.P.A = 10");
                        else if (avg >= 80)
                        printf("C.G.P.A = 9");
                        else if (avg >= 70)
                        printf("C.G.P.A = 8");
                        else if (avg >= 60)
                        printf("C.G.P.A = 7");
                        else if (avg >= 50)
                        printf("C.G.P.A = 6");
                        else if (avg >= 40)
                        printf("C.G.P.A = 5");
                        else
                        printf("fail");
                        return 0;
                        }
