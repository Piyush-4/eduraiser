Program to print all the ASCII values and their equivalent characters
Eduraiser!

Problem:-

Write a program to print all the ASCII values and their equivalent characters using a while loop. The ASCII values vary from 0 t0 255.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators
  • C Loops

The program to print all the ASCII values and their equivalent characters

                    
                        #include <stdio.h>
                        int main() 
                        {
                        int i = 0;
                        while (i <= 255)
                        {
                        printf ("\n%d = %c", i, i);
                        i++;
                        }
                        return 0;
                        }
