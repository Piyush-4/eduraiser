Problem:-

Write a program in C to check that a given year is leap year or not.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators
  • C Conditional

The program to check that a given year is leap year or not

                    
                        #include <stdio.h>
                        int main() {
                        int yr;
                        printf("\nEnter a year: ");
                        scanf("%d", &yr);
                        //Checking that a given year is leap year or not
                        if (yr % 100 == 0) 
                        {
                        if (yr % 400 == 0)
                        printf("Leap year\n");
                        else
                        printf("Not a Leap year\n");
                        }
                        else
                        {
                        if (yr % 4 == 0)
                        printf("Leap year\n");
                        else
                        printf("Not a leap year\n");
                        }
                        return 0;
                        }

Output

                    
                        Enter a year: 2021
                        Not a leap year
