Problem:-

Write a program in C to add two integers.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to add two integers.

                #include <stdio.h>
                    int main() {    
                    
                        int number1, number2, sum;
                        
                        printf("Enter two integers: ");
                        scanf("%d %d",  &number1, &number2);
                    
                        // calculating sum
                        sum = number1 + number2;    
                        
                        printf("%d + %d = %d", number1,number2, sum);
                        return 0;
                    }

Output

                    Enter two integers: 2
                        5
                        2 + 5 = 7
