Problem:-

If a five-digit number is input through the keyboard, write a program to calculate the sum of its digits.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to find the sum of digits of a five digit number.

                    
                        #include <stdio.h>
                        int main() {
                        int sum, d1, d2, d3, d4, d5;
                        long int num;
                        printf("Enter a five digit number:- ");
                        scanf("%ld", &num);
                        //Calculating sum of digits
                        d5 = num % 10;
                        num = num / 10;
                        d4 = num % 10;
                        num = num / 10;
                        d3 = num % 10;
                        num = num / 10;
                        d2 = num % 10;
                        num = num / 10;
                        d1 = num % 10;
                        sum = d5 + d4 + d3 + d2 + d1;
                        printf("The sum of digits of the number entered is %d", sum);
                        return 0;
                        }

Output

                    Enter a five digit number:- 54381
                        The sum of digits of the number entered is 21
