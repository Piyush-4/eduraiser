Program to reverse a five-digit number
Eduraiser!

Problem:-

Write a program in C to reverse a five digit number given by the user.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to reverse a five digit number.

                    
                        #include <stdio.h>
                        int main() {
                        int num, d1, d2, d3, d4, d5;
                        long int revnum;
                        printf("Enter a five digit number(less than 32767):- ");
                        scanf("%d", &num);
                        //Calculating reverse
                        d5 = num % 10;
                        num = num / 10;
                        d4 = num % 10;
                        num = num / 10;
                        d3 = num % 10;
                        num = num / 10;
                        d2 = num % 10;
                        num = num / 10;
                        d1 = num % 10;
                        revnum = d5*10000 + d4*1000 + d3*100 + d2*10 + d1;
                        printf("The reversed number is %ld", revnum);
                        return 0;
                        }

Output

                    Enter a five digit number(less than 32767):- 23456
                        The reversed number is 65432
