Program to convert cartesian co-ordinates into polar co-ordinates
Eduraiser!

Problem:-

Write a program to receive Cartesian co-ordinates(x, y) of a point and convert them into polar co-ordinates (r, θ).
Hint: r = sqrt (x2 + y2) and θ = tan-1 (y/x).

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to convert cartesian co-ordinates into polar co-ordinates

                    
                        #include <stdio.h>
                        #include <math.h>
                        #define PI 3.14285 
                        int main() 
                        {
                        float r, x, y, theeta;
                        printf("Enter the Cartesian co-ordinates (x co-ordinate first followed by y co-ordinate)\n");
                        scanf("%f %f", &x, &y);
                        //Converting cartesian co-ordinates into polar co-ordinates
                        r = sqrt(x * x + y * y);
                        theeta = atan(y/x);
                        theeta = theeta * (180.0/PI);
                        printf("The polar co-ordinate is (%f, %f)", r, theeta);
                        return 0;
                        }
