Problem:-

Write a program in C to find area and perimeter of a rectangle, the length and breadth of the rectangle is entered by the user.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to find area and perimeter of a rectangle.

                    
                        #include <stdio.h>
                        int main() {
                        int length, breadth, area, per;
                        printf("Enter the length and breadth:- ");
                        scanf( "%d %d", &length, &breadth);
                        //Calculating area and perimeter
                        area = length*breadth;
                        per = 2*(length + breadth);
                        printf("Area of the Rectangle = %d \nPerimeter of the rectangle = %d", area, per);
                        return 0;
                        }

Output

                    Enter the length and breadth:- 9
                        3
                        Area of the Rectangle = 27
                        Perimeter of the rectangle = 24
