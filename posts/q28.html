Problem:-

Write a program in C to find the largest number among the three numbers using logical operator.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators
  • C Conditional
  • C Logical operator

The program to find largest number among the three numbers using logical operator.

                    
                        #include <stdio.h>
                        int main() {
                        int a, b, c;
                        printf("Enter three numbes:- ");
                        scanf("%d %d %d", &a, &b, &c);
                        //Finding largest number among the three numbers using logical operator
                        if (a > b && a > c) 
                        printf("%d is largest", a);
                        else if (b > c)
                        printf("%d is largest", b);
                        else
                        printf("%d is largest", c);
                        return 0;
                        }

Output

                    Enter three numbes:- 45
                        76
                        34
                        76 is largest
