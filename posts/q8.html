Problem:-

Write a program in C to find area of a triangle, the height and base of the triangle is entered by the user.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to find area of the triangle.

                    
                        #include <stdio.h>
                        int main() {
                        int height, base;
                        float area;
                        printf("Enter the height and base of the triangle:- ");
                        scanf("%d %d", &height, &base);
                        //Calculating area of the triangle
                        area = 0.5*height*base;
                        printf("Area of the triangle = %f", area);
                        return 0;
                        }

Output

                    Enter the height and base of the triangle:- 3
                        5
                        Area of the triangle = 7.500000
