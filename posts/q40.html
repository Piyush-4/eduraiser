Program to obtain the reversed number of a five-digit number and to determine whether the original and reversed numbers are equal or not

Problem:-

A five-digit number is entered through the keyboard. Write a program to obtain the reversed number and to determine whether the original and reversed numbers are equal or not.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators
  • C Conditional

The Program to obtain the reversed number of a five-digit number

                    
                        #include <stdio.h>
                        #include <conio.h>
                        int main() 
                        {
                        int d1, d2, d3, d4, d5;
                        long int n, num1, num2;
                        printf("\nEnter a five-digit integer Number:- ");
                        scanf("%ld", &num1);
                        printf("\nThe entered five-digit number is:- %ld", num1);
                        n = num1;
                        d5 = num1 % 10;
                        num1 = num1 / 10;
                        d4 = num1 % 10;
                        num1 = num1 / 10;
                        d3 = num1 % 10;
                        num1 = num1 / 10;
                        d2 = num1 % 10;
                        num1 = num1 / 10;
                        d1 = num1;
                        num2 = d5 * 10000 + d4 * 1000 + d3 * 100 + d2 * 10 + d1;
                        printf("\nThe reversed five-digit number is :- %ld", num2);
                        if (n == num2)
                        printf("\nThe original and reversed numbers are equal");
                        else
                        printf("\nThe original and reversed numbers are not equal");
                        getch();
                        return 0;
                        }
