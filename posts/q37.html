Program to count positive, negative and zero numbers
Eduraiser!

Problem:-

Write a program to enter numbers till the user wants. At the end it should display the count of positive, negative and zeros entered.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators
  • C Conditional
  • C Looping

The Program to count positive, negative and zeros Entered

                    
                        #include <stdio.h>
                       int main() 
                        {
                        int p=0, n=0, z=0, num, count=1;
                        while (count == 1)
                        {
                        printf("Enter the Number you want:-\n");
                        scanf("%d", &num);
                        if (num > 0)
                        p++;
                        else if (num == 0)
                        z++;
                        else if (num < 0)
                        n++;
                        else
                        printf("Wrong Entry");
                        printf("Do you want to continue:- \nEnter 1 for yes or 0 for No\n");
                        scanf("%d", &count);
                        }
                        printf("Count of:-\nPositive Numbers = %d\nNegative Numbers = %d\nZeros = %d", p, n, z);
                        return 0;
                        }
