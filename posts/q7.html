Program to find area and perimeter of square
Eduraiser!

Problem:-

Write a program in C to find area and perimeter of a square, the side of square is entered by the user.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to find area and perimeter of the square.

                    
                        #include <stdio.h>
                        int main() {
                        int side, area, per;
                        printf("Enter the side of Square:- ");
                        scanf("%d", &side);
                        //Calculating area and perimeter
                        area = side*side;
                        per = 4*side;
                        printf("Area of the Square = %d \nPerimeter of the Square = %d", area, per);
                        return 0;
                        }

Output

                    Enter the side of Square:- 7
                        Area of the Square = 49 
                        Perimeter of the Square = 28
