Problem:-

Write a program in C to check that an entered character is an alphabet, digit or special symbol.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators
  • C Conditional
  • C Logical operator

The program to check that an entered character is an alphabet, digit or special symbol. 

                    
                        #include <stdio.h>
                        int main() {
                        char ch;
                        printf("Enter a character:- ");
                        scanf("%c", &ch);
                        //Checking that an entered character is an alphabet, digit or special symbol
                        if (ch >= 65 && ch <= 90 || ch >= 97 && ch <= 122) 
                        printf("The entered character is an Alphabet");
                        else if (ch >= 48 && ch <= 57)
                        printf("The entered character is a digit");
                        else if (ch >= 0 && ch <= 47 || ch >= 58 && ch <= 64 || ch >= 91 && ch <= 96 || ch >= 123 && ch <= 126)
                        printf("The entered character is a special symbol");
                        return 0;
                        }

Output

                    Enter a character:- e
                        The entered character is an Alphabet
