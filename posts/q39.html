Program to interchange values of two variables
Eduraiser!

Problem:-

Two numbers are input through the keyboard into two locations. C and D. Write a program to interchange the contents of C and D.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to interchange the values of two variables.

                    
                        #include <stdio.h>
                        #include <conio.h>
                        int main() 
                        {
                        int a, C, D;
                        printf("\nEnter number is C:-  ");
                        scanf("%d", &C);
                        printf("\nEnter number is D:-  ");
                        scanf("%d", &D);
                        printf("The value in C = %d and The value in D = %d", C, D);
                        //Interchanging values of variables
                        a = C;
                        C = D;
                        D = a;
                        printf("\nNew value in C = %d and New value in D = %d", C, D);
                        getch();
                        return 0;
                        }

Output

                    Enter number is C:- 44

                        Enter number is D:- 93
                        The value in C = 44 and The value in D = 93
                        New value in C = 93 and New value in D = 44
