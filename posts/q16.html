Problem:-

Temperature of a city in Fahrenheit degrees is input through the keyboard. Write a program to convert this temperature into Centigrade degrees.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to convert Temperature.

                    
                        #include <stdio.h>
                        int main() {
                        float fah, cel;
                        printf("Enter the temperature in Fahrenheit degrees:- ");
                        scanf("%f", &fah);
                        //Converting temperature from fahrenheit to Centigrade
                        cel = (fah-32) * 5 / 9;
                        printf("The Temperature in Centigrade degrees is:- %f", cel);
                        return 0;
                        }

Output

                    Enter the temperature in Fahrenheit degrees:- 55.7
                        The Temperature in Centigrade degrees is:- 13.166667
