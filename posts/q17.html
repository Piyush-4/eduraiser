Program to find area of a triangle
Eduraiser!

Problem:-

If lengths of three sides of a triangle are input through the keyboard, write a program to find the area of the triangle.

To understand this program, you should be familiar with the following C programming topics:

  • C Input Output (I/O)
  • C Data Types
  • C Variables, Constants and Literals
  • C Operators

The program to find the area of the triangle.

                    
                        #include <stdio.h>
                        #include <math.h>
                        int main() {
                        float a, b, c, sp, area;
                        printf("Enter sides of a triangle:- ");
                        scanf("%f %f %f", &a, &b, &c);
                        //Calculating area of the triangle
                        sp = (a + b + c) / 2;
                        area = sqrt(sp*(sp-a)*(sp-b)*(sp-c));
                        printf("The area of triangle =  %f", area);
                        return 0;
                        }

Output

                    Enter sides of a triangle:- 4
                        5
                        6
                        The area of triangle = 9.921567
